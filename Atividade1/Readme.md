# Introdução à Segurança Ofensiva

Atividade educacional realizada na plataforma TryHackMe (Learn).

## Metodologia
Para explorar as vulnerabilidades da aplicação web simulada pela plataforma, a ferramenta Gobuster foi utilizada via terminal da máquina virtual. Com isso, foi possível listar diretórios ocultos que deveriam ser acessados apenas por membros internos da instituição fictícia. Dessa forma, a página de transferências ficou acessível publicamente, permitindo a realização de uma operação financeira no ambiente simulado.

### Ferramentas
- Máquina virtual com SO Kali Linux (fornecida pela plataforma)
- Gobuster para enumeração de diretórios
